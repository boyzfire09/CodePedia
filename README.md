CodePedia 

CodePedia adalah website sederhana yang berisi pengertian beberapa bahasa pemrograman populer seperti Java, Python, PHP, Dll.
Project ini dibuat menggunakan HTML, TailwindCSS3 untuk tampilan yang modern dan responsif.

🚀 Features

* Menampilkan pengertian beberapa bahasa pemrograman
* Menggunakan TailwindCSS3
* Desain sederhana dan mudah dipahami

🛠️ Tech Stack

- HTML
- TailwindCSS3
